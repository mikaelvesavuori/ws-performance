# Workshop: Varje millisekund (kan) räknas – Prestandabudget, critical rendering path och debugging
Att veta vad användaren vill ha är inte alltid lätt – däremot vet vi att användare i princip aldrig uppskattar väntetid! God prestanda är alltid en team effort men det är ofta upp till frontendare att vara först på bollen. Genom att bättre förstå hur man beräknar och mäter prestanda, skapar en budget, hanterar critical rendering path, förstår objektiv och upplevd laddtid, samt hur man konkret skalar bort onödig väntetid så är vi allihop på väg mot en bättre webb.

## Mål
Förstå moderna tekniker för att öka prestanda
